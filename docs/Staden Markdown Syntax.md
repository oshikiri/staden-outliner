- ## Available Syntaxes
	- Link
		- Raw URL: https://example.com/
		- Link with title: [Example Domain](https://example.com/)
	- Inline code: `content`
	- Code block
		- ```javascript
		  array.map(x => f(x))
		  ```
	- Quote
		- >quote content
	- Heading
		- ## Heading title
	- Pageref
		- [[index]]
	- Task
		- TODO aaaa
		- DOING bbbb
		- DONE cccc
	- Properties
		- key::  value
